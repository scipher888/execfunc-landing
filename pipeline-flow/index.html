How a question flows through the pipeline

Your prompt enters once, then moves top-to-bottom through optional layers that refine the question, gather independent thinking, and return a clearer final judgment.

Entry point

The user asks one question, then chooses any combination of the three pipeline tools.

What can be selected?

You can run one tool, two tools, or the full stack. The system adapts the route based on what was requested.

Stage 1

🧭 Mode detection colors the run

A preprocessor reads the question and auto-detects the operating mode. That mode shapes every later step — what gets asked, how analysis happens, and how synthesis is written.

Stage 2 · Optional

💡 Max Prompt sharpens the question

If requested, the system enriches the prompt before any deeper analysis. It asks targeted clarifying questions based on the active mode, then produces a stronger prompt.

Light = 1 question
Standard = 2–3
Deep = progressive
Output = sharpened prompt
Mode-specific extraction examples
🔧 M2 BuilderEvidence gaps, constraints, prior attempts, missing assumptions.
🌿 M3 WisdomValues in conflict, what “good” feels like, the human stakes beneath the question.
⚖️ M4 TensionWhich pole you lean toward, what each side protects, what gets lost by resolving too fast.
↓
Stage 3 · Optional

⚖️ Council runs independent parallel analysis

If requested, the prompt is sent down parallel paths. Each member works independently. The mode shapes how each one analyzes, but they do not talk to each other.

Critical rule: every member receives the same prompt independently. No cross-talk. No shared drafts. No consensus-building until synthesis.

Option B

🎯 Switching modes on the fly

Your default mode stays set across conversations — usually 🔧 Builder (M2). But for any single request, you can override it inline by naming the mode in the prompt.

Important: inline mode override only affects that one run. It does not change your saved default.
🧲 “Council this in persuasion mode: [question]”Runs council in M1. Your default still stays M2.
🌿 “Max prompt, council, compute this in wisdom mode: [question]”Runs the full pipeline in M3.
⚖️ “Compute this in tension mode: [question]”Runs solo max compute in M4.
